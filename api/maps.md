# Maps API

Map assets, pins, and temporal map projection.

**Prerequisite:** [Campaign model](../architecture/campaign-model.md)

---

## Map model

Maps are wiki pages with map asset metadata and pins targeting entity pages. Temporal projection controls pin visibility by campaign date and discovery state.

Maps are **world-history views** — not tactical battle simulators.
