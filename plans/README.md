# Cursor Plan archives

This folder holds **archived Cursor Plan-mode** documents migrated from the pre–`Esiana-ttrpg` monorepo layout (`GIT\esiana`).

## How to use

- **Roadmap source of truth:** [`esiana-core/todo.md`](../../esiana-core/todo.md)
- **Active plans (editable in Plan mode):** `%USERPROFILE%\.cursor\plans\` on your machine
- **Archives here:** read-only reference for phase history, design decisions, and completed todos

These files are not application runtime documentation. They are agent-generated planning artifacts kept in the wiki repo for version control and team visibility.

## Migration

Archived on 2026-05-30 when the project moved to:

- `Esiana-ttrpg/esiana-core` — application
- `Esiana-ttrpg/docs` — how-to wiki (this repo)
- `Esiana-ttrpg/community-plugins` — plugin hub

Path references inside plans were rewritten from `GIT\esiana` to `Esiana-ttrpg\esiana-core` (and `docs` where applicable).
